# multi-gen-sequencing-benchmark

Simulate and compare multiple sequencing technologies вЂ” Illumina short-read, Oxford Nanopore R9/R10 long-read, PacBio HiFi, and hybrid assemblies вЂ” against the *Escherichia coli* K-12 MG1655 reference genome across 5Г—/10Г—/20Г—/30Г—/50Г— coverage. The current Part C benchmark contains 55 real assembly records evaluated with QUAST, including repeat/plasmid stress tests.

## Quickstart

Run the full 5-step pipeline from the project root:

```bash

```

Output is logged to `logs/pipeline.log`. Steps can also be run individually (see below).

## Pipeline steps

| Step | Script / tool | Output |
|------|--------------|--------|
| 1 | `scripts/simulate_illumina.py` | `data/reads/illumina_30x_{1,2}.fastq` |
| 2 | `badread simulate` (Nanopore profile) | `data/reads/nanopore_30x.fastq` |
| 3 | `badread simulate` (random error model, 99.5 % identity) | `data/reads/hifi_30x.fastq` |
| 4 | `scripts/stat_reads.py` | `results/tables/read_stats.csv` |
| 5 | `scripts/plot_read_lengths.py` | `results/figures/read_length_distribution.png` |

### Step 1 вЂ” Illumina simulation

Pure-Python HiSeq 2500 paired-end simulator (no external tools required).

```bash
python scripts/simulate_illumina.py \
    --reference data/reference/ecoli.fasta \
    --coverage 30 \
    --read-length 150 \
    --insert-mean 350 \
    --insert-sd 50 \
    --output-prefix data/reads/illumina_30x_
```

### Steps 2вЂ“3 вЂ” Long-read simulation (badread)

```bash
# Nanopore ONT
badread simulate \
    --reference data/reference/ecoli.fasta \
    --quantity 30x \
    --length 10000,5000 \
    > data/reads/nanopore_30x.fastq

# PacBio HiFi
badread simulate \
    --reference data/reference/ecoli.fasta \
    --quantity 30x \
    --length 15000,3000 \
    --identity 99.5,99.9,0.5 \
    --error_model random \
    --qscore_model random \
    > data/reads/hifi_30x.fastq
```

### Step 4 вЂ” Read statistics

```bash
python scripts/stat_reads.py
```

Writes `results/tables/read_stats.csv` and prints a summary table.

### Step 5 вЂ” Read length distribution plot

```bash
python scripts/plot_read_lengths.py
```

Writes `results/figures/read_length_distribution.png` (300 DPI).
